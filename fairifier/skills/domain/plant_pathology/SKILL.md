# Plant Pathology Metadata Guidance

Focus on host species, pathogen species, isolate identifiers, inoculation design, disease scoring, and sampling timepoints.

Prefer metadata that helps FAIR-DS package selection:
- host organism / NCBI taxonomy
- pathogen organism / isolate
- plant tissue or organ sampled
- inoculation method and biosafety notes
- assay platform and replicate structure
