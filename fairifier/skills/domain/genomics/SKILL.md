# Genomics Metadata Guidance

Emphasize sample provenance, sequencing platform, library strategy, read/assembly identifiers, and accession-style metadata.

Look for:
- organism and taxonomy
- platform, library prep, read length
- assay type (WGS, RNA-seq, metagenomics, amplicon)
- accession numbers or repository references
- replicate design and timepoints
