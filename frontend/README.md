# FAIRiAgent Frontend

React + TypeScript + Vite frontend for the current Web UI.

## Install

```bash
npm install
```

## Develop

```bash
npm run dev
```

Default dev server: `http://localhost:5173`

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

For the normal full application flow, start the project from the repository root with:

```bash
python run_fairifier.py webui
```
